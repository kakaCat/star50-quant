## 📊 模型架构

### 自编码器设计

```

```

- 输入维度: 50 (股票数)
- 隐性因子数: 10
- 隐藏层: [64, 32]
- 批次大小: 32

## ⚠️ 模型问题与限制

### 主要问题

1. **系统性风险过高（99.46%）**
   - 说明科创50股票高度同质化
   - 板块效应极强，个股差异被压缩
   - 选股难度大，Alpha空间有限

2. **因子高度相关（平均0.61）**
   - 10个因子并非独立
   - 存在冗余，实际有效维度可能<10
   - 建议PCA降维或正交化

3. **特质风险极低（0.54%）**
   - 个股独立性弱
   - 基本面差异未能充分体现
   - 可能需要更长时间窗口或更多特征

4. **数据窗口较短（312天）**
   - 仅覆盖2023-09至2024-12
   - 可能无法捕捉长周期风险特征
   - 建议扩展到完整5年数据

### 模型限制

- **线性假设**: 自编码器虽然非线性，但风险分解仍基于线性模型
- **静态模型**: 因子暴露假设为常数，未考虑时变性
- **缺乏经济解释**: 隐性因子无法对应具体市场因素（如行业、市值、成长）

## 🎯 应用场景

### 1. 组合优化

**风险矩阵构建**:
```
Σ = B @ F @ B^T + D

其中:
- B: 因子暴露矩阵 [50, 10]
- F: 因子协方差矩阵 [10, 10]
- D: 特质风险对角矩阵 [50, 50]
```

**用途**:
- cvxpy组合优化的风险约束
- 最小化组合方差
- 风险预算分配

### 2. 风险归因

**收益分解**:
```
R_portfolio = Σ(w_i * β_i * f) + Σ(w_i * ε_i)
             系统性收益        特质收益
```

**用途**:
- 分析收益来源
- 评估因子暴露
- 调整风险偏好

### 3. 压力测试

**情景分析**:
- 因子冲击: 某个风险因子+2σ
- 相关性冲击: 因子相关性上升
- 波动率冲击: VIX飙升场景
