# 指标计算修复说明

## 问题发现

在初始实现中，发现了三个指标计算错误：

### 1. ✅ IC (Information Coefficient) - 正确
```python
ic = np.corrcoef(预测值, 实际值)[0, 1]
```
**含义**: 预测值与实际收益率的Pearson相关系数  
**状态**: ✅ 计算正确，无需修改

---

### 2. ❌ IR (Information Ratio) - 已修复

**错误实现**:
```python
ir = ic_mean / ic_std  # 这是IC的稳定性指标，不是真正的IR
```

**正确实现**:
```python
# 超额收益的均值 / 超额收益的标准差（年化）
excess_returns = 策略收益
excess_mean = np.mean(excess_returns)
excess_std = np.std(excess_returns)
periods_per_year = 252 / window_size
ir = (excess_mean * periods_per_year) / (excess_std * np.sqrt(periods_per_year))
```

**含义**: 衡量单位风险的超额收益能力

---

### 3. ❌ 年化收益 - 已修复

**错误实现**:
```python
annual_return = np.mean(returns_sim) * 252  # 没考虑复利效应
```

**正确实现**:
```python
# 考虑复利的年化收益
cumulative_return = np.prod(1 + returns_sim) - 1
n_days = len(returns_sim) * window_size
annual_return = (1 + cumulative_return) ** (252 / n_days) - 1
```

**含义**: 基于复利的年化回报率

---

### 4. ❌ 最大回撤 - 已修复

**错误实现**:
```python
# 只返回一个平均收益值，无法计算回撤
return np.array([strategy_return])  # 只有1个点！
```

**正确实现**:
```python
# 使用滚动窗口生成时间序列
for i in range(0, len(predictions), window_size):
    window_return = ...
    returns_series.append(window_return)
return np.array(returns_series)  # 返回时间序列

# 基于时间序列计算回撤
cumulative = (1 + returns_sim).cumprod()
running_max = np.maximum.accumulate(cumulative)
drawdown = (cumulative - running_max) / running_max
max_drawdown = np.min(drawdown)
```

**含义**: 策略净值从峰值到谷值的最大跌幅

## 技术细节

### 策略模拟方法
当前使用滚动窗口方法：
- **窗口大小**: 50个样本
- **选股方式**: 每个窗口选top 20%
- **权重**: 等权配置
- **再平衡**: 每个窗口

这是一个**简化策略**，主要用于调参时快速评估。真实策略需要：
- 考虑交易成本
- 流动性约束
- 市场冲击
- 涨跌停限制

### 完整回测的必要性
调参阶段的简化策略只是初步评估，必须在完整回测框架中验证：
